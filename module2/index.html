Competitive programming websites

HackerRank

This website can be used by anyone for interview prep or competitive programming. This is not a hard core competitive coding website like codeforces and topcoder. This has best UI among all other online coding websites and platform. They provide their own customizable ide for the users.

Codechef

This website provides many type of contest for division one and two. This site has different types of online coding competitions . Like long challenges , short challenges as well ,that are known as cookoff and lunchtime contests. This website provide ratings for their users which can help them evaluates their coding skills.

Codeforces

Codeforces is a website made for hard core competitive programmers. This consists of basically three types of divisional contest. For division one two and three. This coding consist of variety of problems on data structures and algorithm and advance mathematics. This is mainly owned and regulated by russian programmers.
